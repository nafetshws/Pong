# Pong

## How to run:
### Linux
```
cd Pong;
make;
./run;
```
### Windows
```
cd Pong;
make windows;
./run
```

## What is this?
This is a simple pong game written in c. You will be able to play single player against a bot or local multiplayer against one of your friends. Have fun!

## libaries:
- GLFW
- GLAD
- CGLM
